# Roadmap

## Now

- Deterministic meeting intelligence extraction.
- Decision and action item detection.
- Owner inference from attendees.
- CRM update and follow-up drafting.
- Approval-first integration command model.
- GitHub, Jira and webhook adapter interfaces.
- Docker Compose local stack.

## Next

- Persistent Postgres storage for meetings, commands, approvals and audit events.
- Queue-backed dispatch worker for approved integration commands.
- HubSpot and Salesforce production adapters.
- Slack and Microsoft Teams notification adapters.
- OpenAI-compatible LLM extraction adapter with privacy controls.
- Dashboard approval queue and audit view.

## Later

- Multi-tenant organisations and RBAC.
- SSO/SAML/OIDC.
- SOC 2 aligned operational controls.
- Enterprise data retention policies.
- On-prem/private cloud deployment profile.
- Evaluation harness for extraction accuracy.
